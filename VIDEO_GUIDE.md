# 📹 Execution Video Guide

## Recording Your Demo Video (Max 5 Minutes)

### What to Include

Your video should demonstrate:

1. **Introduction (15 seconds)**
   - Show the live URL
   - Brief intro to Pear Media
   - Show features at a glance

2. **Text Enhancement Workflow (1 minute)**
   - Enter text prompt (e.g., "A serene beach at sunset with palm trees")
   - Click "Analyze & Enhance"
   - Show entities detected
   - Show sentiment analysis
   - Show enhanced prompt
   - Click "Approve & Generate"

3. **Image Generation Workflow (2 minutes)**
   - Show auto-filled enhanced prompt
   - Click "Generate/Analyze Images"
   - Wait for images to generate
   - Show generated image variations
   - Explain the results

4. **Optional: Image Upload (1 minute)**
   - Upload a sample image
   - Show image analysis
   - Generate variations from the uploaded image

5. **Conclusion (15 seconds)**
   - Show a few generated images
   - Thank you/outro
   - API credits

### Recording Setup

**Tools:**
- OBS Studio (Free)
- ScreenFlow (Mac)
- Camtasia
- Built-in screen recorder

**Settings:**
- Resolution: 1080p or 720p
- FPS: 30fps
- Audio: Clear microphone
- Duration: 3-5 minutes max

### Recording Steps

1. **Open Pear Media Live Link**
   ```
   https://pear-media.vercel.app
   OR
   http://localhost:3000
   ```

2. **Test All Features**
   - Text input works
   - Enhancement works
   - Image generation works
   - No errors in console

3. **Record Screen**
   - Start recorder
   - Navigate slowly
   - Pause between steps
   - Read descriptions

4. **Save & Export**
   - Export as MP4
   - Name: `pear-media-demo.mp4`
   - Size: Keep under 200MB

### Quality Tips

✅ **DO:**
- Speak clearly
- Pause between steps
- Show results clearly
- Zoom in if needed
- Use mouse highlights

❌ **DON'T:**
- Rush through features
- Show API keys
- Include errors without explanation
- Make it too long
